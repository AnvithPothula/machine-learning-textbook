# Chapters

This textbook is organized into 12 chapters covering 200 concepts in machine learning.

## Chapter Overview

1. [Introduction to Machine Learning Fundamentals](01-intro-to-ml-fundamentals/index.md) - Introduces core ML terminology, paradigms, data concepts, features, labels, models, and algorithmsâ€”establishing the foundation for all subsequent learning.

2. [K-Nearest Neighbors Algorithm](02-k-nearest-neighbors/index.md) - Covers the intuitive KNN algorithm including distance metrics, k-selection, decision boundaries, and applications to classification and regression.

3. [Decision Trees and Tree-Based Learning](03-decision-trees/index.md) - Explores decision tree structure, splitting criteria, pruning, overfitting/underfitting concepts, and feature space partitioning.

4. [Logistic Regression and Classification](04-logistic-regression/index.md) - Introduces logistic regression with sigmoid functions, binary and multiclass classification, and probabilistic interpretations.

5. [Regularization Techniques](05-regularization/index.md) - Covers regularization methods to prevent overfitting including L1/L2 regularization, Ridge and Lasso regression.

6. [Support Vector Machines](06-support-vector-machines/index.md) - Comprehensive coverage of SVMs including hyperplanes, margins, kernel trick, and various kernel types.

7. [K-Means Clustering and Unsupervised Learning](07-k-means-clustering/index.md) - Explores k-means clustering with centroids, initialization strategies, and cluster evaluation methods.

8. [Data Preprocessing and Feature Engineering](08-data-preprocessing/index.md) - Covers essential preprocessing including normalization, standardization, encoding, and feature engineering techniques.

9. [Neural Networks Fundamentals](09-neural-networks/index.md) - Comprehensive introduction to neural networks including architecture, activation functions, backpropagation, and gradient descent.

10. [Convolutional Neural Networks for Computer Vision](10-convolutional-networks/index.md) - Covers CNNs including convolution operations, pooling layers, and famous architectures like ResNet and VGG.

11. [Transfer Learning and Pre-Trained Models](11-transfer-learning/index.md) - Explores transfer learning concepts including fine-tuning, feature extraction, and domain adaptation.

12. [Model Evaluation, Optimization, and Advanced Topics](12-evaluation-optimization/index.md) - Comprehensive coverage of evaluation metrics, cross-validation, optimization techniques, and hyperparameter tuning.

## How to Use This Textbook

Progress through the chapters sequentially, as each chapter builds on concepts from previous chapters. The dependency structure ensures that prerequisite knowledge is always covered before more advanced topics. Each chapter includes:

- Conceptual explanations with mathematical foundations
- Practical implementation examples using Python
- References to Jupyter notebooks in the Code folder
- Exercises and real-world applications

---

**Note:** Each chapter includes a complete list of concepts covered. Make sure to master the fundamentals in early chapters before advancing to neural networks and deep learning topics.
