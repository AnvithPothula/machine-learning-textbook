Understanding Network Architecture

Depth (number of layers) allows the network to learn hierarchical features. Deeper networks can represent more complex functions.

Width (neurons per layer) determines the network's capacity at each level. Wider layers can capture more features simultaneously.

Parameters are the learnable weights and biases. More parameters mean greater capacity but risk overfitting.
